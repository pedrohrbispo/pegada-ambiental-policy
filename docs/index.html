TERMO DE USO DO APLICATIVO PEGADA AMBIENTAL

1. Aceitação dos Termos

O presente Termo de Uso estabelece as regras para utilização do aplicativo Pegada Ambiental.

Ao acessar ou utilizar o aplicativo, o usuário declara que leu, compreendeu e concorda integralmente com estes Termos de Uso. Caso o usuário não concorde com qualquer disposição deste documento, recomenda-se que não utilize o aplicativo.

2. Objetivo do Aplicativo

O aplicativo Pegada Ambiental tem como objetivo promover a conscientização ambiental por meio do monitoramento de indicadores relacionados à sustentabilidade.

A plataforma permite que usuários e instituições registrem e acompanhem dados ambientais, como:

  • Consumo de energia elétrica
  • Consumo de água
  • Uso de papel
  • Produção e gestão de resíduos
  • Práticas de reciclagem
  • Ações de sustentabilidade

Essas informações podem ser utilizadas para geração de relatórios, indicadores ambientais e rankings entre instituições, incentivando boas práticas ambientais.

3. Cadastro de Usuários

Para acessar determinadas funcionalidades do aplicativo, poderá ser necessário realizar um cadastro.

Durante o processo de cadastro, o usuário poderá fornecer informações como:

  • Nome
  • E-mail
  • Instituição ou escola vinculada

O usuário se compromete a fornecer informações verdadeiras, completas e atualizadas.

Cada usuário é responsável por manter a confidencialidade de suas credenciais de acesso.

4. Responsabilidades do Usuário

O usuário compromete-se a utilizar o aplicativo de maneira ética e legal, sendo proibido:

  • Inserir informações falsas ou enganosas
  • Tentar acessar dados de outros usuários sem autorização
  • Realizar tentativas de invasão ou ataques ao sistema
  • Utilizar o aplicativo para fins ilícitos

Caso sejam identificadas violações destes termos, a conta do usuário poderá ser suspensa ou removida.

5. Propriedade Intelectual

Todo o conteúdo presente no aplicativo, incluindo:

  • Layout
  • Interface
  • Código-fonte
  • Textos
  • Imagens
  • Funcionalidades

é protegido por direitos de propriedade intelectual e pertence aos responsáveis pelo projeto Pegada Ambiental, salvo quando indicado de outra forma.

É proibida a reprodução, modificação ou distribuição do conteúdo sem autorização prévia.

6. Limitação de Responsabilidade

As informações apresentadas no aplicativo possuem caráter educativo e informativo.

Os indicadores e resultados exibidos são estimativas baseadas nos dados fornecidos pelos usuários e não devem ser considerados como avaliações ambientais oficiais.

Os responsáveis pelo aplicativo não se responsabilizam por decisões tomadas exclusivamente com base nas informações disponibilizadas pela plataforma.

7. Alterações nos Termos de Uso

Os presentes Termos de Uso podem ser atualizados a qualquer momento, visando melhorias no serviço ou adequação à legislação vigente.

Recomenda-se que os usuários revisem periodicamente este documento para se manterem informados sobre eventuais alterações.

8. Legislação Aplicável

Este Termo de Uso é regido pelas leis da República Federativa do Brasil, especialmente pelas disposições relacionadas ao uso da internet e proteção de dados.

POLÍTICA DE PRIVACIDADE

Esta Política de Privacidade descreve como o aplicativo Pegada Ambiental coleta, utiliza, armazena e protege os dados dos usuários, em conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018 – LGPD).

1. Coleta de Dados

O aplicativo pode coletar dois tipos principais de informações:

Dados de Identificação — Informações fornecidas pelo usuário durante o cadastro, como:

  • Nome
  • E-mail
  • Instituição ou escola

Dados Ambientais — Informações inseridas pelos usuários relacionadas às práticas ambientais, como:

  • Consumo de energia
  • Consumo de água
  • Uso de papel
  • Produção de resíduos
  • Práticas de reciclagem
  • Ações sustentáveis desenvolvidas pela instituição

2. Finalidade do Tratamento dos Dados

Os dados coletados são utilizados para:

  • Calcular indicadores de sustentabilidade
  • Gerar relatórios ambientais
  • Apresentar rankings entre instituições participantes
  • Promover educação ambiental e conscientização ecológica
  • Melhorar a experiência dos usuários no aplicativo

3. Base Legal para Tratamento de Dados

O tratamento de dados realizado pelo aplicativo baseia-se nas hipóteses legais previstas na LGPD, incluindo:

  • Consentimento do titular dos dados
  • Legítimo interesse para fins educacionais e de sustentabilidade

4. Compartilhamento de Dados

Os dados coletados não são comercializados nem compartilhados com terceiros, exceto quando necessário para:

  • Cumprimento de obrigações legais
  • Determinação judicial
  • Execução de políticas públicas ou projetos educacionais

5. Armazenamento e Segurança

Os dados coletados são armazenados em ambientes digitais protegidos.

Medidas de segurança são adotadas para proteger as informações contra:

  • Acesso não autorizado
  • Perda de dados
  • Alteração indevida
  • Vazamento de informações

Essas medidas incluem controles de acesso, autenticação de usuários e boas práticas de segurança da informação.

6. Direitos do Titular de Dados

De acordo com a Lei Geral de Proteção de Dados (LGPD), os usuários possuem os seguintes direitos:

  • Confirmar a existência de tratamento de dados
  • Acessar seus dados pessoais
  • Solicitar correção de dados incompletos ou desatualizados
  • Solicitar anonimização ou exclusão de dados
  • Revogar consentimentos concedidos anteriormente

As solicitações podem ser feitas por meio dos canais de contato do projeto.

7. Retenção dos Dados

Os dados serão armazenados apenas pelo período necessário para cumprir as finalidades educacionais e operacionais do aplicativo.

Após esse período, os dados poderão ser:

  • Excluídos permanentemente
  • Anonimizados para uso estatístico

8. Contato

Em caso de dúvidas, solicitações ou exercício de direitos relacionados à proteção de dados, o usuário poderá entrar em contato com os responsáveis pelo projeto Pegada Ambiental por meio dos canais oficiais da aplicação.
